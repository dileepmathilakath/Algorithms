# Asymptotic Notations
```
O Big oh - Upper bound
Ω Big omega - Lower bound
Θ Theta - Average bound
```
### Examples on computing notations

```
f(n) = n^2logn + n
n^2logn <= n <= 10n^2logn
O(n) = n^2logn
Ω(n) = n^2logn
Θ(n) = n^2logn
```
```
f(n) = n! = n*(n-1)*...*1
1*1*1*...*1 <= n! <= n*n*n*..*n
1 <= n! <= n^n
O(n) = n^n
Ω(n) = 1
Θ(n) = no theta here
```
