# Comparison of functions

## Apply Log on both sides

```
Formulas:

Log ab = Log a + Log b
Log a/b = Log a - Log b
Log a^b = bLoga
a^LogcB = bLogCA
```

```
Eg f(n) = n^2Logn   g(n) = n(logn)^10

Apply Log for  f(n) and g(n)

Log[n^2Logn]
Logn n^2 + LogLogN
2LogN + LogLogN

Log(n(logn)^10)
Log n + Log(Logn)^10
Logn + 10 LogLogn

so f(n) > g(n)
```
